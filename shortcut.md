| 删除     | 描述                                     |
|:---------|:-----------------------------------------|
| d i 符号 | 删除匹配的符号内部的内容，不包含符号本身 |
| d a 符号 | 删除匹配符号内的内容，包含符号本身       |

| 移动 | 描述          |
|:-----|:--------------|
| 0    | 行首          |
| ^    | 行首符号处    |
| $    | 行尾符号处    |

| tab页 | 描述          |
|:------|:--------------|
| b h   | 切换右侧tab页 |
| b l   | 切换左侧tab页 |
| b b   | 首字母切换tab |
|       |          |

| 窗口 | 描述          |
|:-----|:--------------|
| C-d  | 下翻半页      |
| C-u  | 上翻半页      |
| z z  | 当前行居中    |
| w o  | 切换其他窗口  |
